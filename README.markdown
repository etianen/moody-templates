moody-templates
===============

**moody-templates** is a fast and powerful template engine for Python 3.

    {% for name in ["world", "dave",] %}
        Hello {{name}}
    {% endfor %}


Features
--------

*   Easy to use.
*   High performance.
*   Autoescaping.
*   Template inheritance.
*   Supports native python expressions.
*   Extensible template macros.
